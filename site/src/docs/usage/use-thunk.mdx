title: useThunk
---

`useThunk` enables action creators to return a function.
Inspired from [redux-thunk](https://github.com/reduxjs/redux-thunk).

## Usage

```jsx
import { useThunk } from 'middleware-hooks';

const App = () => {
  const [store, dispatch] = useThunk(reducer, initialState);
  return <h1>Hello World!</h1>;
};
```

### Sample Action creators

```js
function incrementIfOdd() {
  return (dispatch, getState) => {
    const { counter } = getState();

    if (counter % 2 === 0) {
      return;
    }

    dispatch(increment());
  };
}

const onButtonClick = () => dispatch(incrementIfOdd());
```
