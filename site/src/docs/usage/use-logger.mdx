title: useLogger
---

`useLogger` will log the action and the associated state.
Inspired from [redux-logger](https://github.com/LogRocket/redux-logger).

## Usage

```jsx
import { useLogger } from 'middleware-hooks';

const App = () => {
  const [store, dispatch] = useLogger(reducer, initialState);
  return <h1>Hello World!</h1>;
};
```
